# 皇家小虎快递匹配指南

## 使用方法
程序分为erp上传与trd上传两个模块
1. 将目标表erp/trd拖入到程序的"文件上传与合规校验"区域，等待分析结果
2. 分析无误后，点击"数据库入表与比对分析"，等待比对结果

### 注
erp与trd要分别建模:
- erp的原则是数据越多越全越好 
- trd的原则是与erp匹配的越多越好
