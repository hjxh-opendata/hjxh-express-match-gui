## TRD 文件格式规范

1. TRD 文件的文件名需要以 **trd\_** 作为前缀
2. TRD 文件的文件格式为：**.xlsx | .xls**
3. TRD 文件中必须要有一张"明细"表，且表名以"**\_明细**"开头
4. TRD 表中至少需要包含以下字段：
   1. 快递公司（价格表对应的公司抬头）
   2. 快递单号
   3. 收货地区
   4. 实际重量
   5. 发货时间
   6. 实际价格

:::warn

1.  文件名 ≠ 表名
2.  TRD 表比 ERD 多了"实际价格"字段
3.  ERP 表中的"快递公司"是现成的，但 TRD 可能需要手动拉填
    :::
